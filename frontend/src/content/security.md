# セキュリティポリシー

最終更新日: 2025年2月

## 1. 認証

exe.dev プロキシベースの認証を使用。パスワードは保存しません。

## 2. 通信の暗号化

すべての通信は exe.dev プロキシを通じた HTTPS で保護されます。

## 3. データ保存

SQLite データベースは exe.dev VM 上に保存。保存時の暗号化は行っていません。

## 4. AI 処理

インタビューデータは Anthropic Claude API に HTTPS 経由で送信されます。

## 5. セッション分離

ユーザーは自分のセッションのみアクセス可能。共有は明示的な操作が必要です。

## 6. 保存しない情報

パスワード、決済情報は一切保存しません。

## 7. 脆弱性の報告

セキュリティに関する問題を発見した場合は [GitHub Issues](https://github.com/susumutomita/DeepForm/issues) までご報告ください。

## 8. オープンソース

ソースコードは公開されており、誰でも監査可能です。
