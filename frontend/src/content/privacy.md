# プライバシーポリシー

最終更新日: 2025年2月

## 1. 収集する情報

- **認証情報**: exe.dev の「Login with exe」を通じたメールアドレスおよびユーザー ID
- **インタビューコンテンツ**: ユーザーが入力したテキストおよび AI 生成物（ファクト・仮説・PRD・仕様書）
- **セッションデータ**: 作成日時、更新日時、共有設定

## 2. 情報の利用目的

- ユーザー認証およびセッション識別
- インタビュー内容の AI 処理（Anthropic Claude API を使用）
- セッション履歴の保存および表示
- キャンペーン機能を通じた他ユーザーとの共有

## 3. 第三者へのデータ提供

インタビュー内容は AI 処理のため Anthropic Claude API に送信されます。それ以外の第三者へのデータ提供は行いません。

## 4. データ保存

データは exe.dev VM 上の SQLite データベースに保存されます。ユーザーがセッションを削除するまで保持されます。

## 5. Cookie

DeepForm 自体は Cookie を使用しません。認証は exe.dev プロキシが管理します。言語設定とテーマ設定は localStorage に保存されます。

## 6. ユーザーの権利

- セッションデータの閲覧・削除
- アカウントの削除要求（GitHub Issues より）

## 7. お問い合わせ

[GitHub Issues](https://github.com/susumutomita/DeepForm/issues) までご連絡ください。
