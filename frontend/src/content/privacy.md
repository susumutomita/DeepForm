# プライバシーポリシー

最終更新日: 2026年2月

## 1. 収集する情報

- **認証情報**: GitHub OAuth を通じたメールアドレス、表示名、アバター URL
- **インタビューコンテンツ**: ユーザーが入力したテキストおよび AI 生成物（ファクト・仮説・PRD・仕様書）
- **セッションデータ**: 作成日時、更新日時、共有設定
- **アクセスログ**: IP アドレス、User-Agent、リファラー、閲覧ページ、UTM パラメータ
- **位置情報（国レベル）**: IP アドレスからローカルデータベース（MaxMind GeoLite2）を用いて国コードを推定します。外部サービスへの IP アドレスの送信は行いません。

## 2. 情報の利用目的

- ユーザー認証およびセッション識別
- インタビュー内容の AI 処理（Anthropic Claude API を使用）
- セッション履歴の保存および表示
- キャンペーン機能を通じた他ユーザーとの共有
- サービス改善のためのアクセス解析（国別アクセス数、ページビュー数、コンバージョン率等の集計統計）

## 3. 第三者へのデータ提供

インタビュー内容は AI 処理のため Anthropic Claude API に送信されます。それ以外の第三者へのデータ提供は行いません。IP アドレスの国コード変換はサーバー上のローカルデータベースで処理され、外部 API には送信されません。

## 4. データ保存

データは exe.dev VM 上の SQLite データベースに保存されます。ユーザーがセッションを削除するまで保持されます。アクセスログは統計処理後、個人を特定できない形に集計されます。

## 5. Cookie

DeepForm 自体は Cookie を使用しません。認証は GitHub OAuth および exe.dev プロキシが管理します。言語設定とテーマ設定は localStorage に保存されます。

## 6. ユーザーの権利

- セッションデータの閲覧・削除
- アカウントの削除要求（GitHub Issues より）
- アクセスログの削除要求
